📊 API Stock

            
            

                
GET Ottieni Stock Disponibile

                
GET /tables/stock?page=1&limit=100

                
Descrizione: Restituisce l'elenco completo dello stock disponibile con quantità e prezzi medi.

                
Parametri Query:

                
    
                    
  • page - Numero pagina (default: 1)
    • 
                    
  • limit - Elementi per pagina (default: 100)
    • 
                    
  • search - Cerca per codice materiale
    • 
                

                
Risposta:

                
{
  "data": [
    {
      "id": "uuid",
      "codice_materiale": "MP001",
      "quantita_disponibile": 50,
      "prezzo_medio": 12.50
    }
  ],
  "total": 10,
  "page": 1,
  "limit": 100
}

            

            
            

                
GET Ottieni Stock per Codice

                
GET /tables/stock?search=MP001

                
Descrizione: Cerca lo stock di un materiale specifico per codice.

                
Uso: Per verificare disponibilità prima di uno scarico.

            

        

        
        
        

            
📤 API Scarichi

            
            

                
POST Crea Nuovo Scarico

                
POST /tables/scarichi_magazzino

                
Descrizione: Registra un nuovo scarico di materiale da magazzino.

                
Body (JSON):

                
{
  "numero_ordine": "ORD-2024-001",
  "codice_materiale": "MP001",
  "quantita": 10,
  "note": "Scarico per ordine cliente",
  "data_scarico": 1705234567890
}

                
⚠️ IMPORTANTE: Prima di creare uno scarico:

                
    
                    
  1. Verifica lo stock disponibile con GET /tables/stock
    2. 
                    
  2. Controlla che quantita_disponibile >= quantita
    3. 
                    
  3. Dopo lo scarico, aggiorna lo stock con PUT /tables/stock/{id}
    4. 
                

            

            
            

                
GET Ottieni Storico Scarichi

                
GET /tables/scarichi_magazzino?page=1&limit=100

                
Descrizione: Restituisce lo storico di tutti gli scarichi.

                
Filtri disponibili:

                
    
                    
  • search - Cerca per numero ordine o codice materiale
    • 
                    
  • sort - Ordina per campo (es. -data_scarico per più recenti)
    •

💻 Esempio Integrazione JavaScript

            
            

                
Scenario: Sistema esterno vuole scaricare materiali

                
// 1. Verifica stock disponibile
async function verificaStock(codice) {
  const response = await fetch(`/tables/stock?search=${codice}`);
  const data = await response.json();
  return data.data.find(s => s.codice_materiale === codice);
}

// 2. Scarica materiale se disponibile
async function scaricaMateriale(ordine, codice, quantita) {
  // Verifica stock
  const stock = await verificaStock(codice);
  
  if (!stock || stock.quantita_disponibile < quantita) {
    throw new Error('Stock insufficiente');
  }
  
  // Crea scarico
  const scarico = await fetch('/tables/scarichi_magazzino', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      numero_ordine: ordine,
      codice_materiale: codice,
      quantita: quantita,
      data_scarico: Date.now()
    })
  });
  
  // Aggiorna stock
  const nuovaQuantita = stock.quantita_disponibile - quantita;
  await fetch(`/tables/stock/${stock.id}`, {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      codice_materiale: stock.codice_materiale,
      quantita_disponibile: nuovaQuantita,
      prezzo_medio: stock.prezzo_medio
    })
  });
  
  return await scarico.json();
}

// 3. Utilizzo
try {
  const result = await scaricaMateriale('ORD-2024-001', 'MP001', 10);
  console.log('Scarico completato:', result);
} catch (error) {
  console.error('Errore:', error.message);
}

            

        

        
        
        

            
📋 Note per l'Integrazione

            
    
                
  • Tutte le API restituiscono JSON
    • 
                
  • Le date sono in formato timestamp (millisecondi)
    • 
                
  • Gli ID sono UUID generati automaticamente
    • 
                
  • CORS: Le API sono accessibili dal browser (stessa origine)
    • 
                
  • Autenticazione: Non richiesta per le API interne (sistema trusted)
    • 
                
  • Gestione Stock: L'aggiornamento dello stock deve essere gestito dall'applicazione chiamante
    • 
                
  • Calcolo Prezzo Medio: Deve essere calcolato lato client usando la formula del prezzo medio ponderato
    •
