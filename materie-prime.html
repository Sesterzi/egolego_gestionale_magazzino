Importa Materie Prime da CSV/Excel

📋 Formato File Richiesto

Il file CSV o Excel deve contenere le seguenti colonne (con intestazione):

codice,part_number,codice_colore,foto_url,colore

Tutti i campi sono OBBLIGATORI:

  • codice → Codice materiale univoco
  • part_number → Part number prodotto
  • codice_colore → Codice identificativo colore (es. RAL9005)
  • foto_url → URL immagine del materiale
  • colore → Nome colore (es. Nero, Bianco)

Esempio:

codice,part_number,codice_colore,foto_url,colore MAT-001,PN-12345,RAL9005,https://esempio.com/foto1.jpg,Nero MAT-002,PN-12346,RAL9016,https://esempio.com/foto2.jpg,Bianco MAT-003,PN-12347,RAL3020,https://esempio.com/foto3.jpg,Rosso

Note:

  • Usa virgola (,) come separatore
  • Prima riga: intestazione colonne
  • Tutti i 5 campi devono essere compilati
  • "0" è un valore valido per codice_colore (es. sistema di codifica colori)
  • Encoding: UTF-8

ℹ️ Come Funziona l'Import

Il sistema importa solo materiali con codice nuovo:

  • ✅ Codice NON presente → Materiale viene importato
  • ⚠️ Codice già esistente → Riga viene saltata (dati esistenti non modificati)

Esempio: Se hai MAT-001 nel database e lo reimporti, viene saltato. Questo protegge i dati esistenti da sovrascritture accidentali.

Formati supportati: CSV, XLSX, XLS
